# Metroidvania 3D Gameplay System (Unity)

Sistema de mecánicas principales desarrollado en Unity (C#) para un juego tipo Metroidvania 3D.

Este proyecto incluye implementación del controlador del jugador, IA enemiga, trampas del entorno y sistemas de progresión del nivel.

## Tecnologías utilizadas

Unity
C#
NavMesh
Animator Controller
UI Canvas System

## Características principales

Sistema de movimiento del jugador
IA enemiga con detección y ataque
Trampas interactivas del entorno
Sistema de progresión del nivel
Integración de UI y audio

## Mi rol en el proyecto

Desarrollé las mecánicas principales del jugador, diseñé los niveles jugables e implementé sistemas de interacción, enemigos y trampas dentro del entorno.

## Estado del proyecto

Repositorio orientado a mostrar la lógica de gameplay implementada.
